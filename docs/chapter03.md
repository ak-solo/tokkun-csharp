# 3章 分岐

## 基礎知識

### 分岐とは

プログラムは通常、上から順番に実行されます。しかし「点数が 60 点以上なら合格、そうでなければ不合格」のように、**条件によって実行する処理を変えたい**場面があります。これを**分岐**といいます。

```
　　　条件を判定
　　　　　↓
　　┌─────────┐
　　│ 条件が真？│
　　└─────────┘
　　  ↙         ↘
　true のとき  false のとき
　  処理 A      処理 B
```

分岐を使うことで、状況に応じた柔軟なプログラムが書けるようになります。

---

### bool 型と条件式

**bool 型**は `true`（真）か `false`（偽）の 2 値だけを持つ型です。`if` 文の条件部分には bool の値が入ります。

```csharp
bool isPass = true;
bool isFail = false;
```

**条件式**（比較演算子を使った式）を評価すると bool 値になります。

```csharp
int x = 10;
Console.WriteLine(x > 5);   // → True
Console.WriteLine(x > 20);  // → False
Console.WriteLine(x == 10); // → True
```

bool 型の変数に条件式の結果を入れておき、後で使うこともできます。

```csharp
bool isPositive = (x > 0);
if (isPositive)
{
    Console.WriteLine("正の数です");
}
```

---

### if 文

条件によって処理を分岐するには `if` 文を使います。条件が `true` のときだけ `{}` 内の処理が実行されます。

```csharp
if (x > 0)
{
    Console.WriteLine("正の数です");
}
```

`else` を加えると、条件が `false` のときの処理も書けます。

```csharp
if (x >= 60)
{
    Console.WriteLine("合格");
}
else
{
    Console.WriteLine("不合格");
}
```

`else if` で 3 つ以上の分岐も表現できます。条件は上から順に評価され、最初に `true` になったブロックだけが実行されます。

```csharp
if (score >= 80)
{
    Console.WriteLine("優");
}
else if (score >= 70)
{
    Console.WriteLine("良");
}
else if (score >= 60)
{
    Console.WriteLine("可");
}
else
{
    Console.WriteLine("不可");
}
```

> **ポイント:** `else if (score >= 70)` が評価されるのは、すでに `score >= 80` が `false` だとわかっている場合だけです。そのため「70 以上 80 未満」という条件をわざわざ書かなくても正しく動きます。

---

### 比較演算子

条件式で使う演算子です。結果は必ず `bool`（`true` か `false`）になります。

| 演算子 | 意味 | 例 | 結果（x=10 のとき） |
|---|---|---|---|
| `>` | より大きい | `x > 5` | `true` |
| `<` | より小さい | `x < 5` | `false` |
| `>=` | 以上 | `x >= 10` | `true` |
| `<=` | 以下 | `x <= 9` | `false` |
| `==` | 等しい | `x == 10` | `true` |
| `!=` | 等しくない | `x != 10` | `false` |

> **注意:** C# では変数への代入は `=`、比較は `==` と記号が違います。`if (x = 10)` と書くとエラーになります。

---

### 論理演算子

複数の条件を組み合わせるには論理演算子を使います。

| 演算子 | 意味 | 真になる条件 |
|---|---|---|
| `&&` | かつ | 左辺も右辺も `true` |
| `\|\|` | または | 左辺か右辺のどちらかが `true` |
| `!` | ではない | 元の値が `false` |

```csharp
int x = 5;
Console.WriteLine(x > 0 && x < 10);  // → True（0より大きく10より小さい）
Console.WriteLine(x < 0 || x > 3);   // → True（どちらかが成立）
Console.WriteLine(!(x == 5));         // → False（x=5 を反転）
```

`&&` は左辺が `false` なら右辺を評価しません（**短絡評価**）。
`||` は左辺が `true` なら右辺を評価しません（短絡評価）。

```csharp
// x が 0 のとき、y / x はゼロ除算エラーになる
if (x != 0 && y / x > 1)  // x=0 なら右辺を評価しない → エラー回避
{
    Console.WriteLine("OK");
}
```

### switch 文

変数の値に応じて多くの分岐があるときは `switch` が読みやすくなります。

```csharp
switch (month)
{
    case 1: case 3: case 5: case 7: case 8: case 10: case 12:
        Console.WriteLine("大の月");
        break;
    case 2: case 4: case 6: case 9: case 11:
        Console.WriteLine("小の月");
        break;
    default:
        Console.WriteLine("そんな月はありません");
        break;
}
```

`case 値1: case 値2:` のようにまとめて書けます。どの `case` にも当てはまらない場合は `default` が実行されます。各 `case` の末尾には `break;` が必要です。

`if` 文との使い分けの目安：
- 同じ変数を `== 値` で比べるだけなら → `switch`
- 範囲比較（`>=`、`<` など）や複数の変数を使う条件 → `if`

---

### 偶数・奇数の判定

`%` 演算子（余り）を使います。2 で割った余りが 0 なら偶数、それ以外なら奇数です。

```csharp
if (x % 2 == 0)
{
    Console.WriteLine("偶数");
}
else
{
    Console.WriteLine("奇数");
}
```

負の数も同様に判定できます（例: `-4 % 2 == 0` → 偶数）。
